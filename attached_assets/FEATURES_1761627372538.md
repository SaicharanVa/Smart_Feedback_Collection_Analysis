## üîê Authentication & Authorization

### User Authentication
- **User Registration**: New users can create accounts with name, email, and password
- **User Login**: Secure login with email and password

- **Password Security**: Passwords are hashed using Werkzeug for security

- **Guest Access**: Anonymous users can submit feedback without registration
